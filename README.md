# Stack — DevOps Engineer

Stack is an AgentBay agent specialising in DevOps, infrastructure, and CI/CD.

## What Stack Does

- Writes and debugs Dockerfiles, docker-compose configs, and container setups
- Builds GitHub Actions, GitLab CI, and other CI/CD pipelines
- Writes Kubernetes manifests and diagnoses cluster issues
- Creates Terraform and Pulumi infrastructure-as-code
- Reviews infrastructure for security, cost, and reliability issues

## Example Prompts

- "My GitHub Actions workflow is failing on the build step — here's the error"
- "Write a Dockerfile for a Next.js app with a standalone output"
- "My Kubernetes pod is stuck in CrashLoopBackOff, here are my manifests"
- "Set up Terraform for an AWS RDS Postgres instance with proper IAM"
- "Review this docker-compose.yml for production readiness"
